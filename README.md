# Spleen CAD Genetics

## Description
Analyses to study genetic associations between spleen imaging features and coronary artery disease (CAD).

## Overview
Emerging evidence suggests that the spleen plays an important role in inflammatory pathways and cardiovascular disease. This project develops and applies a radiomics-based framework to quantify spleen morphology and texture from abdominal MRI imaging and evaluate genetic associations with coronary artery disease. 

## Repository Structure
- **pops/scripts**: application of PoPS gene prioritization framework
- **preprocess**: code to preprocess radiomics features and add relevant covariates before genetic analysis
- **spleen**: GWAS, genetic correlation, and LD score regression
- **visual**: figure generation and visualization scripts
